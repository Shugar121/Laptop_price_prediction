# Laptop Price Prediction

Это веб-приложение на Flask, которое предсказывает цену ноутбука по его характеристикам.

## Возможности
- Предсказание цены ноутбука по характеристикам
- Отображение первых 10 строк датасета
- Показ метрик модели (R2, MAE)
- Обработка ошибок неправильного ввода
